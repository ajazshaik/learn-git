git commands

git init

git remote add origin "url"

git pull origin master

git status   - to check the files status

git add  "filename"   - this allows to add latest files/folder added

git add -A  - allows you to add all files at once

git commit -a  - allows you to commit all files at once

git log  - shows you logs of files you commited

git commit -m "message"  - this allows you to commit the changes you have done with custom message
